# TRAVELL-INN-JAVA
A console-based Java Travel Booking System using OOP concepts like inheritance, polymorphism, abstraction, exception handling, collections, and the State pattern. Users can sign up, log in, view hotels and transport, book services, make payments, cancel bookings, and admins can update prices.


# Features
User roles: Customer and Admin

Sign Up / Login

View & update profile

Browse hotels by location

Book hotel rooms

Browse transportation options

Book travel tickets

Payment through wallet or UPI

Booking cancellation with refund

Admin-only price updates

State pattern for booking status
